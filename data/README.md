# About Data Set

## [Fisher's Iris Data](http://www.math.uah.edu/stat/data/Fisher.html)

```
Type  PW   PL  SW  SL
0      2   14  33  50
1     24   56  31  67
1     23   51  31  69
0      2   10  36  46
```

- `Type`: 0 is *Setosa*, 1 is *Verginica*, and 2 is *Vesicolor*
- `PW`: petal width
- `PL`: petal length
- `SW`: sepal width
- `SL`: sepal length
